## 転機: Claude Codeとの出会い

3. **実装をClaude Codeと一緒に書く** → テストが通る
4. **リファクタリング** → コードレビューもAIに依頼

## 具体的な活用方法

### 1. CLAUDE.md でプロジェクトルールを定義

プロジェクトのルートに `CLAUDE.md` を置くことで、Claude Codeがプロジェクトの文脈を理解してくれます。

```markdown
# プロジェクト概要
このプロジェクトはXXXを実現するアプリです。

## 技術スタック
- Next.js 14
- TypeScript
- Tailwind CSS

## コーディング規約
- 関数は短く、単一責任で
- テストは必ず書く
```

### 2. 効果的なプロンプトパターン

```text
このエラーを修正して:
[エラーメッセージを貼り付け]

期待する動作:
[どうなってほしいか]
```
